Grandmaly is a Large Language Model AI system with the purpose of fixing languid errors of an English manuscript, such errors including grammatical mistakes, typos, word use, and punctuation. All you have to do is upload a doc or docx file and download the fixed word file!

With an account, you will be provided with the option to subscribe and enjoy extended perks of Grandmaly. You will also be given a 2-week trial period before deciding to subscribe.

Functionalities

Manuscript Upload and Processing

Users can upload their academic manuscripts in various formats, including doc and docx.

The system processes the manuscript, dividing it into paragraphs, and integrates with a Large Language Model API for comprehensive editing.

Special consideration is given to handling citations, especially those generated using EndNote.

Automated Editing

The system submits each paragraph to the Large Language Model API with prompts to address grammar, typographical errors, word usage, and punctuation.

Responses from the API are collected and merged back into the original manuscript.

User Subscription Management

Users can create accounts with different roles, including admin and subscribed users.

A trial period of two weeks is provided for new users to explore the app's functionality.

Subscribed users enjoy access based on their subscription plans.

Payment Integration

The app integrates payment methods such as Stripe and Momo, providing a seamless and secure payment experience.

Subscribed users can manage their subscriptions, ensuring uninterrupted access to the manuscript editing services.

Comparison with Existing Apps

Addresses common issues found in similar apps like Grammarly and Quiltbot, such as high costs and concerns about data privacy and confidentiality.

The focus on academic content and specific features tailored for manuscript editing sets it apart from generic text-based processing tools.
